# DATA-404 Project: AI-Based Medical Diagnostic Support Framework

## Overview
This repository contains the work for the DATA-404 project proposal:  
**AI-Based Medical Diagnostic Support Framework Using Public Health Data.**

The goal of this project is to integrate multimodal public health datasets to estimate risk (likelihood) of selected chronic diseases and present insights through an interactive analytics dashboard.  
This project is **educational and exploratory**, not for clinical use.
